How to add changed code again in github

Previously we needed 6setps to add new code in github

  1. git init
  2. git add .
  3. git commit -m "message"
  4. git branch
  5. git remote address
  6. git push

But for adding edited code we need to perform 3 steps

  1. git add .
  2. git commit -m "message"
  3. git push
